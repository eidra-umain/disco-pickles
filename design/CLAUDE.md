# Design Directory

## Ownership
This directory is owned by the Designer.

## Files
- `design-spec.md` — Design specification document
- `*.pen` — Wireframe files

## Rules
- Design spec must reference the PRD (`pm/prd.md`)
- All wireframes should follow the project's design system
- Design changes require Designer approval
- Use the `design-spec` skill to generate or update specs

## Design Spec Sections
1. Visual Language (colors, typography, spacing)
2. Component Library
3. Screen-by-Screen Breakdown
4. Interaction Patterns
5. Accessibility Requirements
6. Responsive Behavior
